# Prompts Compartidos

Este directorio contiene prompts reutilizables por todos los agentes.

## Uso

Los agentes pueden referenciar estos prompts en sus skills:

```markdown
## Pasos

1. Cargar prompt de: /shared/prompts/viral-title.md
2. Aplicar con el contexto actual
```

## Índice de Prompts

| Prompt               | Descripción                   |
| -------------------- | ----------------------------- |
| `viral-title.md`     | Generación de títulos virales |
| `hook-generator.md`  | Crear ganchos de retención    |
| `seo-description.md` | Descripciones optimizadas     |
