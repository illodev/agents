# Assets - Scripts (Guiones)

Este directorio almacena los guiones generados.

## Nomenclatura

`script-{idea_id}.md`

## Formato

```markdown
---
idea_id: idea-001
title: "Título del video"
duration_target: 55
word_count: 142
created: 2026-01-31T10:00:00Z
status: draft|approved|recorded
---

[HOOK]
Primera frase de impacto...

[CONTEXTO]
Establece el problema...

[DESARROLLO]
Puntos principales...

[REVELACIÓN]
Momento clave...

[CIERRE]
CTA o final abierto.
```
